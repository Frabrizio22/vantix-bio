How to Verify a Janoshik COA

Anyone can create a fake PDF. Understanding how to verify COA authenticity protects your research.

Step 1: Get the Report ID

Every Janoshik COA has a unique Report ID in the top-right corner. Format: DM230415-01

Step 2: Use Janoshik's Database

Go to janoshik.com/tests and enter the Report ID.

Step 3: Cross-Reference Results

The database shows:

  • Test date
  • Product tested
  • Purity results
  • Client name (if public)

If the PDF matches the database exactly → verified. If anything differs → fake.

Red Flags

  • Report ID not found in database
  • Purity numbers don't match
  • Test date differs
  • Product name doesn't match

Bottom line: Never trust a COA PDF alone. Always verify through Janoshik's database.
