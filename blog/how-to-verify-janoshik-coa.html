How to Verify a Janoshik COA: 7 Authentication Steps

Verification 6 min read April 21, 2026
TL;DR: Janoshik provides an official verification system at janoshik.com/verification. Enter the sample ID from your COA to confirm authenticity. Red flags: missing batch numbers, reports that can't be verified in their database, generic COAs used across multiple products, or impossible test results (>100% purity). When in doubt, email Janoshik directly with the report ID.

Why COA Verification Matters

The research peptide market has a trust problem. Because these materials are sold for Research Use Only (RUO)—outside pharmaceutical regulation—quality assurance falls entirely on the researcher.

Third-party certificates of analysis (COAs) from labs like Janoshik Analytical have become the de facto standard for peptide verification. But that trust creates an incentive: fake COAs.

Some vendors:

  • Photoshop test results to show higher purity
  • Reuse a single COA across multiple batches (or products)
  • Provide partial reports missing critical data like batch numbers
  • Manufacture entirely fake PDFs with Janoshik's branding

The good news: Janoshik makes verification straightforward. Their public database allows anyone to confirm a COA's authenticity without involving the vendor.

The Official Verification Method

Step 1: Locate the Sample ID

Every legitimate Janoshik COA includes a unique sample ID or batch reference number. This is typically an alphanumeric code (e.g., "F005B", "TIRZ-2024-0127") prominently displayed on the report.

Red flag: COA has no sample ID, batch number, or report identifier. Without this, verification is impossible.

Step 2: Go to the Official Verification Page

Navigate to janoshik.com/verification. Do not use any other domain claiming to be Janoshik—phishing sites exist.

Step 3: Enter the Sample ID

Use the search or verification function to input the sample ID from your COA. The system will return the original laboratory record for that batch, including:

  • Compound name
  • Purity percentage (HPLC)
  • Molecular weight confirmation (LC-MS/MS)
  • Test date
  • Endotoxin level (if tested)

Step 4: Cross-Check the Data

Compare the values on your vendor-provided COA against Janoshik's database entry:

Field What to Verify
Compound Name Exact match (e.g., "Tirzepatide", not "Tirz")
Purity Should match to 0.1% (e.g., 99.2% = 99.2%)
Molecular Weight Must match expected m/z for the peptide
Test Date Should be recent (within 3-6 months for most suppliers)

If any value differs—even by 0.5%—the COA has been altered.

Step 5: Check Batch Number Match

The batch number on your product label must match the batch number on the COA. This confirms you received the tested material.

Red flag: Vendor provides a COA but won't share the batch number, or the batch number on your vial doesn't match the report. This suggests the same COA is being reused across multiple batches.

7 Red Flags of Fake or Suspicious COAs

1. Report Not Found in Janoshik Database

The most obvious sign. If the sample ID returns "no results" or "not found" on janoshik.com/verification, the COA is either fabricated or altered.

2. Missing Batch Number or Sample ID

A legitimate testing lab assigns every sample a unique identifier. COAs without this cannot be verified and should be treated as worthless.

3. Generic COA Used Across Multiple Products

Some vendors show the same COA for Tirzepatide, Retatrutide, and Semaglutide. Each peptide is structurally different and would produce unique LC-MS results. This is impossible.

How to check: Request COAs for 2-3 different products. If the sample IDs, test dates, or purity values are identical, walk away.

4. Perfect Results Across All Products

If every peptide tests at exactly 99.0%, 99.5%, or 100.0%, question it. Real manufacturing has variance. Legitimate suppliers see ranges like 98.7%, 99.2%, 99.6%—not uniform perfection.

Especially suspicious: >100% purity (thermodynamically impossible).

5. Inconsistent Fonts, Logos, or Formatting

Janoshik's official reports follow a consistent template. Compare your COA to examples on their website or from verified suppliers.

Look for:

  • Misaligned text
  • Low-resolution logos
  • Spelling errors ("Janoschik", "Janoshek")
  • Inconsistent date formats (MM/DD/YYYY vs DD-MM-YYYY switching mid-report)

6. Vendor Refuses to Provide COA Before Purchase

Legitimate suppliers publish COAs openly or provide them upon request before purchase. If a vendor says COAs are "confidential," "available after order," or charges extra for documentation, that's a red flag.

7. Old or Undated Test Results

COAs should reflect recent testing. If the test date is:

  • >6 months old: Ask if newer testing exists
  • >12 months old: Question whether this batch is still being sold
  • Missing entirely: Cannot verify freshness or batch relevance

Peptides degrade over time, especially GLP-1 agonists. A 2-year-old COA doesn't guarantee current quality.

What to Do If Verification Fails

Option 1: Email Janoshik Directly

Janoshik will confirm COA authenticity if you provide the report ID. This is the gold standard if you're uncertain.

Contact: Use the contact form at janoshik.com or email them with the sample ID in question.

Option 2: Request a New COA from the Vendor

If the batch number doesn't match or the COA can't be verified, ask the vendor for:

  • A COA specific to your batch (not a generic report)
  • The exact sample ID used in Janoshik's database
  • Clarification on any discrepancies

A legitimate vendor will provide this immediately. A dishonest one will deflect, delay, or refuse.

Option 3: Find a Different Supplier

If verification fails and the vendor can't explain why, don't gamble. The research peptide space has vendors with verifiable, batch-specific COAs. Use one.

Why Batch-Level Verification Matters

Even honest vendors can have batch-to-batch variance. Purity can shift 2-3 percentage points between production runs due to:

  • Raw material quality differences
  • Synthesis scaling issues
  • Storage or handling conditions

A single "master COA" doesn't tell you what's in your vial. Batch-specific testing does.

Example: Vendor shows a Tirzepatide COA at 99.4% purity. But that was Batch A from January. You receive Batch C from March, which tested at 96.8%. Without batch-level verification, you'd never know you're underdosing by 2.6%.

Final Verification Checklist

Before purchasing from any peptide supplier, confirm:

✓ COA has a unique sample ID or batch reference number

✓ Sample ID can be verified at janoshik.com/verification

✓ Batch number on product label matches COA

✓ Test results match Janoshik's database (purity, molecular weight, date)

✓ COA is recent (within 6 months preferred)

✓ Vendor provides COAs before purchase, not after

✓ Different products have different sample IDs (not reused reports)

If even one of these fails, ask questions. If the vendor can't provide satisfactory answers, find one who can.
